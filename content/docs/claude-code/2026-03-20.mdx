**投稿内容:**

> Claude CodeにChannels機能が追加されました。TelegramやDiscordを接続して、実行中のセッションに直接メッセージを送ることができます。
> 流れは以下の通りです:
> 1. TelegramやDiscordからメッセージを送信
> 2. そのメッセージがイベントとしてClaude Codeのセッションに到達
> 3. Claudeが内容を処理して何をするかを判断
> 4. あなたの環境で実行
