**投稿内容:**

> VPSで素早くコードを書くためのClaude Codeエイリアスの更新版：
> 
> c() \{ IS_SANDBOX=1 claude --continue --dangerously-skip-permissions "$@"; \}
> 
> --continue はログアウトした場合に前のセッションを継続します。
> 
> 追加方法：
> echo 'c() \{ IS_SANDBOX=1 claude --continue --dangerously-skip-permissions "$@"; \}' >> ~/.bashrc && source ~/.bashrc
> 
> これで~/.bashrcに設定され、ログインするたびに実行され、ログイン後にただ「c」と入力するだけでClaude Codeが起動します！

**解説:**

著名インディーハッカーの@levelsio氏が、VPS（仮想プライベートサーバー）上でClaude Codeを快適に使うためのbash設定を公開した実践Tipsです。1文字のエイリアス「c」で起動できるようにし、セッション継続フラグ`--continue`と危険を承知で権限制限をスキップする`--dangerously-skip-permissions`を組み合わせています。これによりSSH接続が切れても作業を中断せずに続けられ、リモート環境でのAI支援開発効率が大幅に向上します。以前の投稿も引用されており、長期的に洗練されたワークフローを共有する内容で、CLIツールの運用ノウハウとして価値が高いです。
