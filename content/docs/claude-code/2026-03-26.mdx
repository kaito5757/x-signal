**投稿内容:**

> Claude Code 2.1.83がリリースされました。3つのフラグ変更、76のCLI変更、7つのシステムプロンプト変更があります。
> 主なハイライト：
> ・managed-settings.d/というドロップイン用ディレクトリが追加され、ポリシーフラグメントをアルファベット順にマージ可能に。チームごとに別々にデプロイしやすくなりました。

> ・Ignore memory機能がMEMORY.mdを空として扱うようになり、保存されたメモリ行がコンテキストに含まれるのを防ぎます。
> 
> 詳細な変更内容はスレッドに続きます。

**解説:**

AnthropicのCLIツール「Claude Code」の最新バージョン2.1.83に関する詳細な変更ログを投稿したものです。企業利用を強く意識したセキュリティ強化が最大のポイントで、子プロセスへの認証情報漏洩防止は特に重要です。また、チーム開発で便利な設定ファイルのモジュール管理機能も追加されています。76ものCLI変更と多数のバグ修正、UI改善が含まれており、Claude Codeを本格的に業務で使っている開発者にとって非常に実用的なアップデート内容です。セキュリティと運用性の両面で大きな進化を示しています。
