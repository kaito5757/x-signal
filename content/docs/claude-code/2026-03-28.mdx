**投稿内容:**

> Claude Codeがhooksの`if`フィールドをサポートするようになりました。
> 
> これはパーミッションルールの構文を使って、特定のbashコマンドでのみhookを実行するかどうかをフィルタリングできます。すべてのコマンドではなく一部のコマンドだけにhookを適用したい場合に非常に便利です！

**解説:**

Claude Codeチームのメンバーが共有した新機能です。hooksとはClaudeがコマンドを実行する前後に自動で特定の処理（承認、ログ記録など）を行う仕組みですが、これまではすべてのコマンドに対して適用されるため通知過多（approval fatigue）が問題になっていました。新たに追加された`if`フィールドを使うことで「特定のファイル変更時のみ」「特定のコマンドパターンのみ」など条件を指定可能になり、必要な場面だけに安全策を適用できるようになりました。実践的なベストプラクティスとして、開発者のワークフローを大幅に改善するアップデートです。
