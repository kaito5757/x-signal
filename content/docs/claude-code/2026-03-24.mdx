### 3. 読み取り専用ツールの並列化テクニック

| 項目 | 詳細 |

| 投稿日時 | 2026年3月23日 19:02（日本時間） |
| エンゲージメント | ❤️ 245 / 🔁 14 / 💬 27 |
| リンク | [元ポスト](https://x.com/lydiahallie/status/2036156759518445818) |

**投稿内容:**

> Claude Agent SDKで読み取り専用のツールを作成する場合、`readOnlyHint: true` を必ず設定してください。
> 
> これによりClaude Codeに「副作用がない」ことを伝え、安全に並列実行できるようになります。設定しないと他のツールと同時に実行できなくなり、「直列化の障壁」が生じてしまいます。

**解説:**

Anthropicの公式スタッフによる技術Tipsです。Agent SDKを使ってツールを開発する際に重要なパフォーマンス最適化手法を解説しています。readOnlyHintフラグを付けないと、読み取りだけのツールが他のツールの実行をブロックしてしまい、並列処理の利点が失われるという落とし穴を指摘。Claude Codeの並列実行能力を最大限に活かすための必須知識で、Agent開発者にとって非常に実践的です。アニメーション付きで視覚的に分かりやすく説明されています。
